# IT-SIMPLERA-WEEK-02-REPORT-CYBER-SECURITY-ANALYST-INTERN-RED-TEAM
This report covers the fundamentals of computer networking through hands-on practice 
with essential commands. We used ipconfig /all to view network settings, ping to test 
connectivity, tracert to trace data routes, nslookup to find IP addresses, and arp -a to list 
local devices. These tools help identify and fix network problems quickly. 
The report also compares routers and switches, explaining that routers connect different 
networks while switches connect devices within the same network. Additionally, it 
highlights the differences between IPv4 and IPv6 addressing, and describes the seven 
layers of the OSI model with their functions, protocols, and devices, giving a clear picture 
of how network communication works.
